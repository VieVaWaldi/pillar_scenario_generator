# Hi Antonio

## Setup

```
1. git pull https://github.com/VieVaWaldi/pillar_scenario_generator
2. cd pillar_scenario_generator
3. python -m venv venv
4. \venv\Scripts\activate.bat or \venv\Scripts\activate.ps1 (Windows and PowerShell)
5. pip install -r requirements.txt
```

## Run it

`python main.py`
